# chn-site-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面经过整理后统一存放，便于索引、预览和分发。

## 项目简介

chn-site-hub 是一个静态页面集合仓库。它不专注于某个特定网站或域名，而是作为多种独立 HTML 资源的聚合目录。每个页面均保持独立，彼此之间无强依赖关系，可直接在浏览器中打开查看。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（样式、脚本、图片等）
├── index.html      # 仓库首页（可选，用于展示页面列表）
└── README.md       # 本文档
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名通常反映其内容或用途。
- `assets/`：存放页面可能共用的 CSS、JavaScript、字体、图片等资源。如无共用资源，此目录可省略。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，不共享状态或数据。
- 页面名称及内容可能随时间更新或替换，不保证永久兼容。
- 归档内容仅供学习、研究或参考使用，不构成任何形式的保证或承诺。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 欢迎提交 Issue 或 Pull Request 以协助改进页面内容或结构。
- 提交时请保持页面独立性，避免引入外部依赖（除非在 `assets/` 中统一管理）。
- 请勿在页面中包含恶意代码、跟踪脚本或违反法律法规的内容。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/chn-site-hub.git
   ```
2. 直接打开 `pages/` 下的任意 HTML 文件，或在浏览器中启动本地服务器查看。
3. 也可通过 GitHub Pages 等静态托管服务发布整个仓库。
